# Bitcoin Maxi Layout Design

**Date:** 2026-04-07

## Goal

`bitcoin-monolith-quartz-v3`를 기준으로 위키를 다시 배포하되, 정보 구조는 유지하고 화면 경험만 더 강한 "비트코인 맥시" 방향으로 재조정한다.

## Approved Direction

- 레이아웃은 좌측 탐색, 중앙 대형 본문, 우측 대형 그래프 패널 구조를 유지한다.
- 우측 그래프는 작은 보조 위젯이 아니라 실제 탐색 도구처럼 가장 큰 카드로 보이게 한다.
- 디자인 톤은 과한 네온이 아니라 절제된 사이퍼펑크/맥시 감성으로 맞춘다.
- 읽기성을 희생하지 않고 브론즈, 오렌지, 흑갈색, 흑금속 계열의 밀도 있는 대비를 사용한다.

## Layout Decisions

- 좌측 사이드바는 사이트 정체성과 탐색에 집중한다.
- 중앙 본문은 현재보다 더 넓고 깊이감 있는 패널로 만들어 문서 읽기 우선순위를 올린다.
- 우측 사이드바는 그래프를 최상단에 두고, 목차와 백링크는 그 아래 보조 패널로 배치한다.
- 데스크톱에서는 그래프 카드가 충분한 높이를 확보하도록 사이드바 내부에서 가장 큰 영역을 차지하게 한다.
- 모바일에서는 Quartz 기본 적응형 흐름을 유지하되 카드 밀도와 여백을 줄여 한 열로 자연스럽게 접는다.

## Visual Decisions

- 전체 배경은 평면 검정보다 약간의 질감과 깊이가 있는 흑갈색 기반으로 간다.
- 카드 배경은 본문과 사이드바를 모두 묵직한 금속/종이 중간 질감처럼 보이게 한다.
- 강조색은 비트코인 오렌지를 중심으로 하되 황동 계열을 섞어 고급스럽게 보정한다.
- 그래프 패널은 얇은 금속성 보더, 내부 글로우, 강화된 높이로 존재감을 준다.
- 제목, 보조 라벨, 패널 헤더는 터미널 같은 긴장감을 주되 과장하지 않는다.

## Non-Goals

- Quartz 컴포넌트 소스 자체를 포크해서 구조를 바꾸지 않는다.
- 콘텐츠 구조나 문서 링크 체계를 이번 작업에서 새로 설계하지 않는다.
- 과한 애니메이션, 네온 발광, 가독성을 해치는 효과는 넣지 않는다.
